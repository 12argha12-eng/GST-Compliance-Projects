# GST Excel Projects

## Overview

This repository contains two Microsoft Excel projects created to demonstrate practical understanding of GST compliance and Indirect Tax processes. The projects focus on GST registration tracking, compliance monitoring, and Input Tax Credit (ITC) reconciliation using sample business data.

## Projects

### 1. GST Registration & Compliance Tracker
- GST registration tracking
- Registration amendment and cancellation/surrender monitoring
- GSTIN verification
- Compliance status tracking

### 2. GST Input Tax Credit (ITC) Reconciliation Model
- GSTR-2B and GSTR-3B reconciliation
- Vendor-wise ITC comparison
- Tax invoice verification
- XLOOKUP-based vendor mapping
- GST compliance reporting
